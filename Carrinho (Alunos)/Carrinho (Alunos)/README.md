# Carrinho de Compras

## 🛩️ Carrinho de Compras – Miniaturas de Aviões

Projeto desenvolvido durante a disciplina de Linguagem e Desenvolvimento Web no curso Técnico em Informática Integrado ao Ensino Médio – IFSP Caraguatatuba.
Este projeto consiste em uma aplicação web de e-commerce fictício para venda de miniaturas de aviões de guerra, totalmente desenvolvida com HTML, CSS e JavaScript puro, com foco em manipulação do DOM.


## 📖 Sobre o projeto
O objetivo da atividade foi aplicar na prática conceitos fundamentais de JavaScript, criando uma loja virtual funcional com carrinho de compras dinâmico.
A aplicação permite que o usuário:

- visualize produtos
- abra um modal com detalhes
- escolha o tamanho (escala da miniatura)
- adicione itens ao carrinho
- altere quantidades
- visualize subtotal, desconto e valor total automaticamente

Todo o comportamento da página é controlado via JavaScript, sem uso de bibliotecas ou frameworks.

## 🎯 Objetivo acadêmico
Este projeto teve como finalidade consolidar os conhecimentos de:
desenvolvimento front-end
lógica de programação
manipulação de interface com JavaScript
Simulando o funcionamento básico de um carrinho de compras real utilizado em sites de e-commerce.
